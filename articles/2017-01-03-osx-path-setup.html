ตั้งค่า PATH สำหรับ macOS

การใช้งานโปรแกรมประเภท CLI นั้น จำเป็นจะต้องมีการตั้งค่า PATH เพื่อให้ macOS สามารถเรียกใช้โปรแกรม CLI นั้นๆ ได้อยากถูกต้อง ตัวอย่างในบทความนี้จะเป็นค่าตั้งค่า PATH ให้กับโปรแกรม mongoDB เพื่อที่จะได้สามารถเรียกใช้งานจาก Command Line ได้

ตั้งค่า PATH

  1. Copy PATH ที่ต้องการตั้งค่า
  2. เปิดโปรแกรม Terminal
  3. แก้ไขไฟล์ ~/.bash_profile ด้วยโปรแกรม vim โดยการพิมพ์ vim ~/.bash_profile ที่โปรแกรม Terminal
  4. เข้าสู่ Edit Mode โดยการพิมพ์ i
  5. เพิ่ม PATH ที่ด้องการโดยใช้ export PATH=/usr/local/bin:/usr/local/mongodb/bin:$PATH
  6. ออกจาก Edit Mode โดยการพิมพ์ Esc
  7. จัดเก็บ และ ออกจากโปรแกรม vim โดยการพิมพ์ :wq!
  8. ปิด/เปิด โปรแกรม Terminal เพื่อทำการ Reload PATH

จากตัวอย่างเป็นการตั้งค่า PATH ให้กับปรแกรม mongodb

ไฟล์ .bash_profile


export PATH=/usr/local/bin:/usr/local/mongodb/bin:$PATH

ทดสอบการตั้งค่า PATH

พิมพ์คำสั่งที่ Terminal


mongod --version
mongo --version
