พัฒนา Realtime App ด้วย Firebase

Firebase เป็น Project ที่พัฒนาโดย Google ถูกออกแบบมาให้เป็น API และ Cloud Storage สำหรับพัฒนา Realtime App โดยเฉพาะ โดย Firebase นั้นรองรับการทำงานร่วมกับ Modern Platform แทบทุก Platform แต่ในบทความนี้ผู้เขียนจะนำเสนอการพัฒนา Realtime App โดยใช้ Firebase บน Platform ของ JavaScript

ทำความรู้จักกับ Firebase

Firebase เป็น Project ที่พัฒนาโดย Google ถูกออกแบบมาให้เป็น API และ Cloud Storage สำหรับพัฒนา Realtime App** โดยเฉพาะ

Platform ที่รองรับการใช้งาน Firebase

Firebase รองรับการทำงานกับ Platform ยอดนิยมแทบทุก Platform เช่น

  • Angular (มี Library สำหรับ Anuglar โดยเฉพาะชื่อ AngularFire)
  • JavaScript
  • Node.js
  • iOS/OSX
  • Java/Android
  • REST

ซึ่งในบทความนี้ผู้เขียนจะแนะนำวิธีพัฒนา Realtime App โดยใช้ Firebase ร่วมกับการเขียน Code ด้วย JavaScript

สมัครใช้บริการ Firebase

นักพัฒนาสามารถสมัครใช้บริการ Firebase โดยการสมัคร Free Account ได้ที่ firebase.com ซึ่ง Account แบบฟรีจะมีข้อจำกัดที่จำนวน Connection ต้องไม่เกิน 50, Data Transfer ไม่เกิน 5GB และ ความจุของ Storage ไม่เกิน 100MB หลังจากทำการสมัครฟรี Account แล้วนักพัฒนาจะได้รับ URL ของตนเองเช่น https://your-reference.firebaseio.com ซึ่งจะใช้เป็น Reference ในการเชื่อมต่อกับ Firebase API และ เป็น URL ในการเรียกดูหน้า Web สำหรับ Firebase Account ของตนเองอีกด้วย

สร้าง Reference ไปยัง Firebase

ในบทความนี้เพื่อให้ง่ายในการสาธิตการเขียน Code เราจะทำการบันทึกข้อมูลไปยัง Root Path


<script>
    $(function() {
        var firebaseUrl = 'https://your-firebase-xxx.firebaseio.com';
        var commentRef = new Firebase(firebaseUrl).child('comments');

        listenToFirebase();
    });
</script>

การดึงค่าที่ User ป้อนเข้ามาโดยใช้ jQuery

ใช้ jQuery Selector ทำการอ้างอิง HTML ไว้กับตัวแปรทั้ง 3 ตัว


<script>
    $(function() {
        // Code ก่อนหน้านี้
        var input = $('input');
        var button = $('a');
        var ul = $('ul');
    });
</script>

เมื่อกดปุ่ม buttonSubmit เราจะอ่านค่าจาก inputText มาเก็บไว้ในตัวแปรชื่อ text และทำให้ inputText กลายเป็นค่าว่างเพื่อรับค่าในครั้งต่อไป


<script>
    $(function() {
        // Code ก่อนหน้านี้
        button.click(function() {
            var text = input.val();
            input.val('');
        }
    });
</script>

ส่งค่า (Push) ไปยัง Firebase

ในขั้นตอนนี้เราจะส่งค่าในตัวแปรชื่อ text ไปบันทึกไว้ที่ Firebase, ซึ่งเราได้อ้างอิง Account ของ Firebase ผ่านตัวแปรชื่อ firebaseRef


<script>
    $(function() {
        // Code ก่อนหน้านี้
        button.click(function() {
            // Code ก่อนหน้านี้
            firebaseRef.push(text);
        }
    });
</script>

สร้าง Callback เพื่อรับค่าจาก Firebase เมื่อข้อมูลมีการเปลี่ยนแปลง

ทันทีที่ข้อมูลใน Account Firebase มีการเปลี่ยนแปลงจะเกิด Callback นี้ขึ้นมาโดยอัตโนมัติ โดยเราจะใช้วิธีอ่านค่าทั้งหมด (ใหม่-เก่า) แล้วนำมาอัพเดท Unordered List


<script>
    $(function() {
        // Code ก่อนหน้านี้
        function listenToFirebase() {
            commentRef.on('value', function(snapshot) {
                var comments = snapshot.val();
                var i = 1;

                ul.empty();

                if (comments !== null) {
                    $.each(comments, function(index, c) {
                        addToList(parseInt(i) + ': ' + c);
                        i += 1;
                    });
                }
            });
        }

        function addToList(text) {
            var li = '<li class="list-group-item">' + text + '</li>';
            ul.append(li);
        }
    });
</script>

บทสรุป

ในบทความนี้ผู้เขียนได้แนะนำวิธีการการพัฒา Realtime App โดยการใช้ Firebase ร่วมกับการเขียน Code ด้วย JavaScript ซึ่งจะมีข้อแตกต่างกับการใช้งาน Socket.io ของ Node.js ตรงที่นักพัฒนาไม่จำเป็นต้องมี Server เป็นของตัวเอง จึงทำให้การพัฒนา Realtime App นั้นง่ายยิ่งขึ้น

สำหรับการบันทึกข้อมูลไปยัง Firebase นั้นในบทความนี้ได้ยกตัวอย่างเพียงวิธีเดียวคือการใช้ function push() ซึ่งจะใช้กับข้อมูลที่เพิ่มขึ้นเรื่อยๆ แต่หากเป็นการแทนที่ข้อมูลเดิม ก็จะเป็นการใช้งาน function set() และ function update(), ส่วนการลบข้อมูลก็จะเป็นใช้งาน function remove()

การบันทึกข้อมูลไปยัง Firebase นั้นไม่จำเป็นต้องบันทึกไว้ที่ Root Path เสมอไป โดยโครงสร้างการจัดเก็บข้อมูลของ Firebase จะมีความคล้ายกับโครงสร้างของ JSON

Code ตัวอย่างในบทความนี้ถูกทดสอบบน Firebase 1.0.x
