ตั้งค่า PATH สำหรับ Windows

การใช้งานโปรแกรมประเภท CLI นั้น จำเป็นจะต้องมีการตั้งค่า PATH เพื่อให้ Windows สามารถเรียกใช้โปรแกรม CLI นั้นๆ ได้อยากถูกต้อง ตัวอย่างในบทความนี้จะเป็นค่าตั้งค่า PATH ให้กับโปรแกรม Atom Text Editor เพื่อที่จะได้สามารถเรียกใช้งานจาก Command Line ได้เช่น


# เปิดโปรแกรม atom จาก working directory
atom .

# ติดตั้ง atom package ชื่อ fileicons
apm install fileicons

ตั้งค่า PATH

  1. Copy PATH ที่ต้องการตั้งค่า โดยใช้โปรแกรม Windows Explorer (รูป A)
  2. เข้าสู่หน้าจอ System โดยเข้า Control Panel -> System and Security -> System (รูป B)
  3. Click เลือก Advanced system settings (รูป B)
  4. Click เลือก Tab Advanced และ Click ปุ่ม Environment Variables (รูป C)
  5. Click เลือก PATH ในส่วนของ User variables for ... และ Click ปุ่ม Edit (รูป D)
  6. Click ปุ่ม New และ Paste PATH ที่ได้ Copy มาจากข้อ 1. (รูป E)
  7. กรณีมี PATH อื่นๆ เพิ่มเติมให้ทำซ้ำข้อ 6. และ Click ปุ่ม OK
  8. Click ปุ่ม OK เพื่อไล่ปิดหน้าจอที่ได้เปิดไว้ทั้งหมด
  9. ปิด/เปิด โปรแกรม Command Prompt เพื่อทำการ Reload PATH ใหม่ (รูป F)

(รูป A) จากตัวอย่างเป็นการ Copy PATH ของโปรแกรม Atom Text Editor

(รูป B) หน้าจอ System

(รูป C) Tab Advanced

(รูป D) หน้าจอ Environment Varables

(รูป E) หน้าจอ Edit environment variables

(รูป E) กรณี Windows 7 และ 8 ให้ทำการเพิ่ม PATH ต่อท้าย PATH เดิมโดยคั่น PATH แต่ละ PATH ด้วยเครื่องหมาย ;
